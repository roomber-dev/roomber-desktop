### How to compile

Remember to `npm install` before you try it!


In the terminal, use
`npx electron-packager <sourcedir> <appname> --platform=<platform> --arch=<arch> [optional flags...]`

[Used NPM Package](https://www.npmjs.com/package/electron-packager)
